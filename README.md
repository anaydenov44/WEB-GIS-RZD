# WEB-GIS-RZD

**WEB-GIS-RZD** — веб-ориентированная геоинформационная система для анализа железнодорожной инфраструктуры и маршрутов на территории Российской Федерации.

Проект разработан в рамках выпускной квалификационной работы и представляет собой исследовательско-прикладной прототип, объединяющий обработку геоданных, построение железнодорожного графа, работу с маршрутами поездов и интерактивную картографическую визуализацию.

## Назначение проекта

Цель проекта — разработка веб-ГИС, позволяющей:

- отображать железнодорожную инфраструктуру на интерактивной карте;
- работать с данными по федеральным округам Российской Федерации;
- загружать и обновлять геоданные железнодорожной сети;
- сопоставлять маршруты поездов с объектами железнодорожной инфраструктуры;
- строить виртуальные маршруты между станциями;
- выполнять пространственный анализ маршрутных коридоров;
- визуализировать аналитические показатели на карте.

Система не является промышленным продуктом и предназначена для исследовательских, учебных и демонстрационных целей.

## Основные возможности

- Интерактивная карта железнодорожной инфраструктуры.
- Отображение станций, линий и федеральных округов.
- Загрузка и обновление данных по выбранным регионам.
- Поиск и импорт маршрутов поездов.
- Сопоставление остановок маршрута со станциями в базе данных.
- Построение геометрии маршрута по железнодорожному графу.
- Построение виртуального маршрута между двумя станциями.
- Анализ населённых пунктов в коридоре маршрута.
- Расчёт и визуализация аналитических показателей.
- Сравнение альтернативных вариантов маршрута.

## Архитектура проекта

Проект состоит из двух основных частей: backend-приложения и frontend-приложения.

### Backend

Backend реализован на FastAPI и отвечает за:

- подключение к базе данных;
- предоставление REST API;
- загрузку и обновление региональных данных;
- импорт и обработку маршрутов;
- построение железнодорожного графа;
- построение виртуальных маршрутов;
- выполнение аналитических расчётов.

Основной код backend расположен в директории:

```text
back/app/
```

Служебные скрипты для загрузки, нормализации, построения графа и проверки данных расположены в директории:

```text
back/scripts/
```

### Frontend

Frontend реализован на React и отвечает за:

- отображение интерактивной карты;
- выбор федеральных округов;
- отображение станций и железнодорожных линий;
- построение и отображение маршрутов;
- взаимодействие с аналитическими API;
- визуализацию результатов анализа.

Основной код frontend расположен в директории:

```text
front/src/
```
